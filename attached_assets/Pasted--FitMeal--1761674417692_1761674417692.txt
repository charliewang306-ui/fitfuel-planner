📌 产品名称（暂定）
FitMeal（水+营养智能管家）

📌 一句话定义
根据体重与目标，自动规划每天应吃应喝的总量，记录实时更新剩余需求，不会吃也能按推荐补足。

📌 目标用户

想增肌 / 增重 / 减脂 / 减肥的人

对食物营养不了解、不会配餐的人

忙、不爱算、不懂吃的人

📌 核心价值
吃对、喝够、长期坚持。
不懂吃 → 看剩余 → 一键补足 → 达标。

────────────────────────

✅产品整体结构（八大模块）

1️⃣ 每日配额计算引擎（按体重/目标自动算当天需求）

热量 kcal

蛋白 g

脂肪 g

碳水 g

纤维 g

饮水 oz
📌根据体重、活动、目标（日更）自动更新

2️⃣ 记录模块（吃+喝）

搜索添加食物、饮品

显示每份营养：kcal/P/F/C/纤维/钠

支持克、毫升、杯、勺、碗等单位

记录后即时更新进度与剩余

3️⃣ 可视化饮水瓶界面

大瓶子装水升降动画

显示 已喝 / 目标（oz）

快捷 “+8oz +12oz +自定义”

完成进度可打勾

4️⃣ 一键补足推荐引擎（AI推荐）

根据剩余营养差额，给出 2–3 组整套组合

推荐每样食物份量已算好

按钮：一键记录/替换某项/份量±
📌核心差异化功能

5️⃣ 提醒+倒计时系统

吃饭、喝水、训练前/后营养提醒

本地通知：响铃/震动

到点弹出倒计时（默认15分钟）

推迟 15/30/60分钟、完成、跳过

6️⃣ 今日看板（首页）

卡路里圆环进度

蛋白/脂肪/碳水/纤维/饮水条状进度

下方固定栏：剩余差额显示 + 一键补足入口

7️⃣ 时间轴（Timeline）

今日饮食与喝水的待办卡片

动态状态：未开始/倒计时/完成/推迟

8️⃣ 趋势与自动微调

体重折线图

连续7天趋势不符 → 自动提示热量±150–200 kcal

✅页面说明（UI最简5页）
页面	主要功能
1. 今日看板	热量/宏量/水进度、一键补足入口
2. 饮水页面	水瓶可视化、加水按钮、进度条
3. 食物记录页面	列表显示当餐营养数值、添加入口
4. 时间轴提醒页	吃喝计划卡片、倒计时
5. 设置页	作息、目标、单位、提醒参数、过敏偏好等

📌导航方式：底部 Tab（看板、饮水、记录、提醒、设置）

────────────────────────

✅运行逻辑（流程图）

首次使用
录入体重+目标+作息 → 系统算出每日目标 → 进入看板

日常使用
吃东西 → 记录 → 更新进度 → 看剩余 → 一键补足
到点 → 提醒 → 倒计时 → 完成/推迟/跳过 → 更新状态
喝水同理

每天结束
剩余少 → 系统发“收尾推荐”
次日 → 根据体重、完成度智能微调

✅提醒规则（专业默认值）
项目	规则
饮水	0.6oz × lb ；清醒期每60–90min分配；训练前后加大
餐次	目标选择3–5餐；蛋白均匀分配25–40g/餐
夜间	减少水；低脂低碳建议
咖啡因	睡前8小时截止提醒
纤维	晚餐前未达标提示

📌倒计时未完成可自动延期一次

────────────────────────

✅建议引擎规则（核心差异化）

输入：
剩余 P/F/C/kcal

输出：
2–3 套组合推荐（蛋白源+主食±健康脂肪/蔬菜）

约束：

偏差阈值：P±3g、F±2g、C±5g、kcal±50

提供替换项和份量微调

点击“一键记录”立即落账
